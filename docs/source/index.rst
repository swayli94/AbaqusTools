AbaqusTools
=================================

AbaqusTools is a Python library designed to simplify and automate common tasks in Abaqus finite element analysis. 
This toolkit provides a comprehensive set of functions and classes for model creation, meshing, boundary conditions, 
material properties, job management, and post-processing operations.

Key features include:

* Model and part creation utilities
* Automated meshing and boundary condition setup
* Periodic boundary conditions (PBC) implementation
* Composite material modeling with LARC05 failure criterion
* Post-processing and data extraction from ODB files
* User-friendly Python interface for Abaqus scripting

**GitHub Repository:** https://github.com/swayli94/AbaqusTools
